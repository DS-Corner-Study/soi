 원시 자료형: 하나의 값 저장, 크기가 일정한 공간에 저장
 객체 자료형: 여러 개의 값을 저장, 동적인 값 -> 참조 이용(값을 저장한 곳의 주소만 저장)

 - 배열은 객체 자료형에 해당되므로 프로퍼티와 메소드 존재
 1. length 프로퍼티
 2. push 메소드: 배열 마지막 요소 뒤에 값 추가

- 함수도 객체, 값으로 취급
 1. 함수를 값으로 저장하는 함수 표현식 가능, 다른 함수에 인수로 전달 가능
 2. name 프로퍼티: 함수 이름 저장


- 참조에 의한 비교
* 객체는 생성될 때 고유한 참조값을 가짐

let per={
    name:"이정환"
};

let man={
    name:"이정환"
};

 console.log(per===man); //false : 참조값이 다름
